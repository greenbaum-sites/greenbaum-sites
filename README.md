# greenbaum-sites

Four-site portfolio ecosystem for Justin R. Greenbaum. Static HTML, zero build step, deployed on Vercel with custom domains.

| Site | Domain | Role |
|------|--------|------|
| **Justin R. Greenbaum** | justingreenbaum.com | Professional hub — advisory, Coherence, AI governance |
| **Greenbaum Photography** | greenbaumphotography.com | Photography portfolio — Leica, travel, published work |
| **Megabrixels** | megabrixels.com | Creative studio — Lego fine art photography |
| **Greenbaum Labs** | greenbaumlabs.com | AI workshop — DGX Spark, side projects, experiments |

## Deploy

Each subdirectory is a standalone static site. On Vercel, create four projects pointing to the same repo with different root directories. See `COWORK-HANDOFF.md` for full deployment instructions.

## Stack

HTML + CSS + vanilla JS. Google Fonts via CDN. No frameworks. No build step. No dependencies.
